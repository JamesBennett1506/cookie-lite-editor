Privacy Policy for Cookie Lite Editor

Last updated: February 17, 2026

Introduction
Cookie Lite Editor ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our browser extension for Microsoft Edge. By installing and using Cookie Lite Editor, you agree to the practices described in this policy.

We believe in complete transparency. This document details exactly what data the extension accesses, how it is used, and what measures we take to protect your privacy. If you have any questions, please contact us using the information provided at the end of this policy.

1. Information We Access
1.1 Cookie Data
The primary function of Cookie Lite Editor is to view and manage cookies stored by your browser. To perform this function, the extension accesses:

- Cookie names – The identifiers used by websites to store information
- Cookie values – The actual data stored within cookies (may include session tokens, preferences, etc.)
- Cookie domains – Which websites the cookies belong to
- Cookie paths – The specific URL paths where cookies are active
- Expiration dates – When cookies will automatically be deleted
- Security flags – Secure, HttpOnly, and SameSite attributes

1.2 Website Information
To display cookies for the current website, the extension accesses:
- The URL of the active tab
- The domain name of the website you are currently visiting

1.3 Storage
The extension uses Chrome storage APIs only to temporarily cache cookie data for display purposes. No data is permanently stored by the extension itself.

2. How We Use Information
The information accessed by Cookie Lite Editor is used exclusively for:
- Displaying cookies in an organized table format
- Allowing you to edit, add, or delete cookies as you direct
- Exporting cookie data to JSON files (only when you initiate the export)
- Importing cookie data from JSON files (only when you initiate the import)

We do not:
- Collect any personal information
- Transmit any data to external servers
- Track your browsing history
- Store cookies or any data outside your browser
- Share any information with third parties
- Use analytics or tracking tools
- Serve advertisements

3. Data Storage and Transmission
3.1 Local Operations Only
Cookie Lite Editor operates entirely locally within your browser. All cookie operations happen on your device. The extension contains no code that sends data to any remote server, cloud service, or third-party endpoint.

3.2 Export Functionality
When you use the export feature:
- A JSON file is generated locally on your computer
- Your browser's download feature saves this file to your designated download location
- No copy of this file is sent to any server
- You have full control over where and how this file is stored

3.3 Import Functionality
When you use the import feature:
- You select a JSON file from your local device
- The extension reads this file locally
- Cookies are set in your browser based on the file contents
- No data is transmitted during this process

4. Permissions Explained
Cookie Lite Editor requests certain permissions to function. Here is exactly why each permission is needed:

- cookies – To read, create, edit, and delete cookies on your behalf
- activeTab – To determine which website you are currently viewing and display its cookies
- storage – To temporarily cache cookie data for efficient display
- <all_urls> – To access cookies for all websites you visit (cookies are domain-specific)

We request the minimum permissions necessary to provide full cookie management functionality.
